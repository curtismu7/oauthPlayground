# Banking API Postman Collections

This directory contains comprehensive Postman collections for testing the Banking API with both end user and admin user scenarios.

## Structure

- `collections/` - Postman collection files
- `environments/` - Environment configuration files for different deployment targets
- `documentation/` - Setup guides and usage documentation

## Quick Start

1. Import the environment files into Postman
2. Configure OAuth credentials in your chosen environment
3. Import the collection files
4. Follow the setup guide in `documentation/Setup-Guide.md`

## Collections

- **Banking-API-End-User** - Collection for testing end user scenarios with limited scopes
- **Banking-API-Admin** - Collection for testing admin scenarios with full access

## Environments

- **Development** - Local development environment configuration
- **Staging** - Staging environment configuration  
- **Production** - Production environment configuration
