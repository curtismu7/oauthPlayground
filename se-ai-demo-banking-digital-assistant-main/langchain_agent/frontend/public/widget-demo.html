Banking Chat Widget Demo

This page demonstrates how to embed the Banking Chat Widget on any website.

Integration Methods

Method 1: Auto-initialization with data attributes

<!-- Include React (if not already included) --> <script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script> <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script> <!-- Include the widget --> <link rel="stylesheet" href="banking-chat-widget.css"> <script src="banking-chat-widget.js" data-banking-chat-widget data-position="bottom-right" data-theme="light" data-title="AI Banking Assistant" data-auto-open="false" ></script>

Method 2: Manual initialization

<script> // Initialize the widget const widget = window.initBankingChatWidget({ position: 'bottom-right', theme: 'light', title: 'AI Banking Assistant', autoOpen: false, onOpen: () => console.log('Widget opened'), onClose: () => console.log('Widget closed'), onMessage: (message) => console.log('New message:', message) }); // Control the widget programmatically // widget.open(); // widget.close(); // widget.destroy(); </script>

Configuration Options

{ containerId: 'banking-chat-widget', // Container element ID apiUrl: 'ws://localhost:8000/ws', // WebSocket URL position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' theme: 'light', // 'light', 'dark' title: 'AI Banking Assistant', // Widget title autoOpen: false, // Auto-open on load showFAB: true, // Show floating action button fabTheme: 'light', // FAB theme onOpen: function() {}, // Open callback onClose: function() {}, // Close callback onMessage: function(message) {}, // New message callback customStyles: {}, // Custom CSS styles className: '' // Additional CSS class }
