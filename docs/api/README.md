## Documentation Format

Each API reference includes:
- **Overview**: Purpose and use cases
- **API Reference**: Methods, props, types
- **Examples**: Code samples
- **Notes**: Important considerations

## Using the API

All services and components are fully typed with TypeScript. Use your IDE's autocomplete and type checking for the best development experience.
