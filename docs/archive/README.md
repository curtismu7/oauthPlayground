# Archive

Historical documentation, completed session summaries, and deprecated features.

## Organization

- **[2024](2024/)**: Documentation from 2024
- **[2025](2025/)**: Documentation from 2025

## What Goes in Archive

- Completed session summaries
- Old implementation documentation
- Deprecated feature docs
- Historical design decisions
- Superseded guides

## Naming Convention

Use date prefixes for archived files:
- `2025-11-06-feature-name.md`
- `2024-12-15-session-summary.md`

## Retention Policy

- Keep all documentation for at least 1 year
- Archive after feature completion or deprecation
- Maintain for historical reference and learning
