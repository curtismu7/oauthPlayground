AI & OAuth Glossary

Structured glossary covering foundational AI concepts, OAuth & OpenID standards, and the agentic tooling ecosystem. Search or explore each category to learn terminology in context.

Artificial Intelligence (AI)

Simulation of human intelligence in machines encompassing learning, reasoning, perception, and adaptation.

Example Conversational assistants, medical diagnosis systems, recommendation engines.

Related Machine Learning, Deep Learning, Neural Networks

Machine Learning (ML)

Subset of AI where models learn patterns from data to make predictions without explicit programming.

Deep Learning

Uses multi-layer neural networks to learn hierarchical representations for complex tasks like vision and language.

Algorithm

Defined procedure or set of rules that solves a problem or executes a task.

Transformer

Attention-based architecture enabling parallel sequence processing and long-range context understanding.

Example GPT, BERT, and T5 all rely on transformer networks.

Diffusion Model

Generative model that iteratively denoises random noise to synthesize realistic images.

Authorization Code Flow

OAuth 2.0 flow where the authorization server returns a short-lived code that the client exchanges for tokens.

Example Web apps exchanging the code at the backend to keep tokens off the browser.

Access Token

Bearer credential allowing resource access on behalf of a user or client.

ID Token

JSON Web Token issued by OpenID Connect providers containing end-user identity claims.

Retrieval-Augmented Generation (RAG)

Combines semantic retrieval with text generation to ground model outputs in external knowledge.

Workflow Orchestration

Coordinating multi-step agent workflows, tool calls, and human approvals end to end.
