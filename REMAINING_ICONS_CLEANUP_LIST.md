## 🛠️ Fix Pattern

### Step 1: Remove the Import
```typescript
// REMOVE:
import { FiAlertCircle, FiCheckCircle, FiShield } from '@icons';
```

### Step 2: Replace Icon Usage with Emojis
```typescript
// REPLACE:
<FiAlertCircle /> → ⚠️
<FiCheckCircle /> → ✅
<FiShield /> → 🛡️
```

### Step 3: For Styled Components Using Icons
```typescript
// REPLACE:
const Icon = styled(FiAlertCircle)`
  color: red;
`;

// WITH:
const Icon = styled.span`
  color: red;
  &::before { 
    content: '⚠️'; 
  }
`;
```

### Step 4: For Icons with Props
```typescript
// REPLACE:
<FiAlertCircle size={16} />

// WITH:
<span style={{ fontSize: '16px' }}>⚠️</span>
```
