PKCE (Proof Key for Code Exchange)

PKCE is a security extension for OAuth 2.0 that prevents authorization code interception attacks. It's required for public clients and highly recommended for all OAuth flows.

Code Verifier

A high-entropy cryptographic random string (43-128 chars) that stays secret in your app. Think of it as a temporary password that proves you're the same client that started the OAuth flow.
