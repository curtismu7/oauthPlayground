OAuth 2.0 Authorization Code Flow

Most secure OAuth 2.0 flow with PKCE support for public clients and refresh token capability.

Key Features:
  • Access Token + Refresh Token
  • PKCE support for enhanced security
  • State parameter for CSRF protection
  • Backend required for confidential clients

OIDC Authorization Code Flow

Authentication + Authorization flow with ID Token support for user identity verification.

Key Features:
  • Access Token + ID Token + Refresh Token
  • User identity verification
  • SSO capabilities
  • Nonce parameter support

Client Credentials Flow

Machine-to-machine authentication for backend services and microservices.

Key Features:
  • Access Token only
  • No user interaction required
  • Backend service authentication
  • Scope-based access control

Device Authorization Flow

Authentication for input-constrained devices like smart TVs and IoT devices.

Key Features:
  • Polling-based authentication
  • User code for secondary device
  • Short expiration times
  • No keyboard required
